# Forever Order

Users can create and manage Forever Orders or Good Till Triggered orders using this set of APIs.

[REST](#__tabbed_1_1)

<table><tbody><tr><td>POST</td><td>/forever/orders</td><td>Create Forever Order</td></tr><tr><td>PUT</td><td>/forever/orders/{order-id}</td><td>Modify existing Forever Order</td></tr><tr><td>DELETE</td><td>/forever/orders/{order-id}</td><td>Cancel existing Forever Order</td></tr><tr><td>GET</td><td>/forever/orders</td><td>Retrieve an array of all existing forever orders</td></tr></tbody></table>

Order Placement, Modification and Cancellation APIs requires Static IP whitelisting - [here](/docs/v2/authentication/#setup-static-ip)

## Create Forever Order

This API helps you create a new Forever Order.

[cURL](#__tabbed_2_1)

```
    curl --request POST \
    --url https://api.dhan.co/v2/forever/orders \
    --header 'Accept: application/json' \
    --header 'Content-Type: application/json' \
    --header 'access-token: ' \
    --data '{Request JSON}'
```

## Modify Forever Order

Users can take use of this API to make changes to already created forever orders. The variables that can be modified are price, quantity, order type, disclosed quantity, trigger price & validity.

[cURL](#__tabbed_5_1)

```
curl --request PUT \
--url https://api.dhan.co/v2/forever/orders/{order-id} \
--header 'Accept: application/json' \
--header 'Content-Type: application/json' \
--header 'access-token: adad' \
--data '{Request JSON}'
```

  
**Requested Structure**

[cURL](#__tabbed_6_1)

```json
{
  "dhanClientId": "1000000132",
  "orderId": "5132208051112",
  "orderFlag": "SINGLE",
  "orderType": "LIMIT",
  "legName": "TARGET_LEG",
  "quantity": 15,
  "price": 1421,
  "disclosedQuantity": 1,
  "triggerPrice": 1420,
  "validity": "DAY"
}
```

## Delete Forever Order

This API lets you delete pending Forever Order using the Order ID.

[cURL](#__tabbed_8_1)

```
curl --request DELETE \
--url https://api.dhan.co/v2/forever/orders/{order-id} \
--header 'Accept: application/json' \
--header 'access-token: {JWT}'
```

  
**Request Structure**

_No Body_

  
**Response Structure**
